Git Subsystem is comprised of *Index(stagin),*Head,*WorkingDirectory.

See tree contents:
git ls-tree -r <branchname>

git-ls-files - Show information about files in the index and the working tree

change to branch git checkout (branchname)

delete the branch: git branch -d feature_x

TO COMMIT THEM: git commit -m "Commit message" (now it is in HEAD, but not in remote repository(main area)

TO propose changes (add it to the Index) use: git add <filename> git add *

PUSHING files:

git push origin master (where MASTER=whatever branch you want to push your changes to.)

??? If you have not cloned an existing repository and want to connect your repository to a remote server, you need to add it with git remote add origin <server> Now you are able to push your changes to the selected remote server
create a working copy local repo: git clone /path/to/repository
REmote copy: it clone username@host:/path/to/repository
